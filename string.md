# 344 翻转字符串 2026.1.12

> 编写一个函数，其作用是将输入的字符串反转过来。输入字符串以字符数组 `s` 的形式给出。
>
> 不要给另外的数组分配额外的空间，你必须**[原地](https://baike.baidu.com/item/原地算法)修改输入数组**、使用 O(1) 的额外空间解决这一问题。

**示例 1：**

```
输入：s = ["h","e","l","l","o"]
输出：["o","l","l","e","h"]
```

**示例 2：**

```
输入：s = ["H","a","n","n","a","h"]
输出：["h","a","n","n","a","H"]
```

### 思路：

这题简单啊，就是一个交换函数套一个循环而已，以下是单指针写法

```c++
class Solution {
public:
    void reverseString(vector<char>& s) {
        int n=s.size();
        for(int i=0;i<n/2;i++){
            char temp=s[i];
            s[i]=s[n-1-i];
            s[n-1-i]=temp;
        }
    }
};
```

但是我这第一版写的不是很优美，更官方一点或者正式一点，应该使用<mark>双指针</mark>来写，一个left指针，一个right指针，不断向中靠，可能这更加‘健壮’一点

```c++
for(int left=0,right=n-1;left<right;left++,right--)//这里终止条件不用写=，因为到中间的时候只能是奇数，就一个数不用再交换了
```

注意：交换是一件很简单的事，所以下次写记得用库函数：

```c++
swap(a,b)//交换ab
```

# 541 翻转字符串ll 2026.1.12

> 给定一个字符串 `s` 和一个整数 `k`，从字符串开头算起，每计数至 `2k` 个字符，就反转这 `2k` 字符中的前 `k` 个字符。
>
> - 如果剩余字符少于 `k` 个，则将剩余字符全部反转。
> - 如果剩余字符小于 `2k` 但大于或等于 `k` 个，则反转前 `k` 个字符，其余字符保持原样。

**示例 1：**

```
输入：s = "abcdefg", k = 2
输出："bacdfeg"
```

**示例 2：**

```
输入：s = "abcd", k = 2
输出："bacd"
```

 ### 思路：

<mark>分段翻转</mark>

```c++
class Solution {
public:
    string reverseStr(string s, int k) {
        int n = s.size();
        for (int i = 0; i < n; i += k * 2) {
            reverse(s.begin() + i, s.begin() + min(i + k, n));
        }
        return s;
    }
};
```

>

> 输入：s = "the sky is blue"

> ```

>
> ```
> 输入：s = "  hello world  "

> 解释：反转后的字符串中不能存在前导空格和尾随空格。
> ```
